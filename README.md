# Cyber Security Awareness Training

Interactive single-page cyber security awareness course built in plain HTML/CSS/JS — no dependencies beyond Google Fonts.

**Live:** https://srameko.github.io/cyber-awareness/

## What's inside

- **8 modules** covering the full awareness curriculum:
  1. Why Security Matters
  2. Understanding Threats
  3. Social Engineering
  4. Password Security (with interactive strength checker)
  5. Multi-Factor Authentication
  6. Data Protection & Sharing
  7. Reporting Security Incidents
  8. Getting Help & Support
- **Knowledge quiz** — 10 questions with instant feedback
- **Completion certificate** with score and time taken

## Features

- **Multilingual** — EN / CS / SK language switcher
- **Themes** — System (follows OS), Dark, Light
- **Progress bar** and timer
- Single `index.html` file, no build step required
